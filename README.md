# Impacto da Chuva na Temperatura de São Carlos

## 📋 Descrição do Projeto

Este projeto analisa a relação entre precipitação pluviométrica e variações de temperatura na região de São Carlos, São Paulo. O estudo utiliza dados climáticos históricos para investigar como eventos de chuva afetam os padrões de temperatura local.

## 🎯 Objetivo

Entender e quantificar o impacto da chuva na temperatura de São Carlos através de análise de dados climáticos de longo prazo (2006-2026).

## 📊 Dados Utilizados

O projeto inclui três conjuntos de dados climáticos da estação A711:

- **dados_A711_H_2006-09-03_2026-06-03.csv** - Dados horários (18+ MB)
- **dados_A711_D_2006-09-03_2026-06-03.csv** - Dados diários (~400 KB)
- **dados_A711_M_2006-09-03_2026-06-03.csv** - Dados mensais (~9 KB)

**Período de cobertura:** 03/09/2006 a 03/06/2026
